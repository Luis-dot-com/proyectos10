El Reto de los 10 Proyectos de JavaScript

Luis Marquez

Proyectos simples de JavaScript para el aprendizaje de la elaboracion de paginas web haciendo uso del localStorage.
Con nuestras anfitrionas:
Trixiel y Felicra

Proyecto 1

Contador inteligente

Un contador inteligente de números es una herramienta digital capaz de registrar el aumento y la reducción de los números ingresados de forma rápida y precisa.

Proyecto 2

Lista de tareas

Guarda tus tareas, actividades y compras de manera virtual y ordenada. Esta función permite un uso ágil y funcional, facilitando la organización diaria y ayudando a mantener un mejor control de las actividades pendientes.

Proyecto 3

Adivina el numero

Compite contra robo Trixiel (la version malvada de trixiel) para ganar y ser el mejor en adivinar numeros, no es un titulo universitario pero supongo que es algo.

Proyecto 4

Calculadora basica

Una calculadora virtual basica es una herramienta digital diseñada para realizar operaciones matemáticas básicas de manera rápida, precisa e intuitiva.

Proyecto 5

Colores aleatorios

Un generador de colores es una herramienta que crea combinaciones y códigos de colores de forma automática, estos colores generados se enseñan en el sistema de la web.

Proyecto 6

Temporizador

Un temporizador web es una herramienta digital que permite medir y controlar el tiempo de manera precisa y sencilla.

Proyecto 7

Generador de contraseñas

Permite generar contraseñas fuertes y difíciles de descifrar, ayudando a proteger cuentas y datos personales con mayor seguridad y confiabilidad.

Proyecto 8

Modo oscuro/claro

El botón de modo oscuro y claro permite cambiar la apariencia visual de la página entre un tema oscuro y uno claro según la preferencia del usuario.

Proyecto 9

Piedra, papel o tijera

En este juego el usuario compite contra Robo Trixiel eligiendo una de las tres opciones disponibles. ¡Acaba ya con esa copia barata de nuestra dinosauria favorita!

Proyecto 10

Galería de Imágenes con Filtros

Esta galería es un espacio visual diseñado para mostrar imágenes de manera organizada y atractiva dentro de una página.
